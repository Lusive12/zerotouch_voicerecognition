# zerotouch_voicerecognition (ProTel)

**ProTel** is an advanced Indonesian speech processing pipeline designed for real-time and batch interactions. It integrates state-of-the-art offline machine learning models for Speech-to-Text (STT), Wake Word detection, and Text-to-Speech (TTS), entirely locally on your machine.

## ✨ Features

- **Real-Time Speech-to-Text (STT) with Wake Word:**
  - Powered by `faster-whisper` for fast and accurate transcription.
  - State-machine architecture with `openwakeword` for hands-free activation (e.g., "Hey Jarvis").
  - Auto-pauses and saves transcriptions to JSON after detecting silence.
- **Batch STT Processing:**
  - Transcribe existing audio and video files effortlessly.
- **Text-to-Speech (TTS):**
  - High-quality, offline Indonesian TTS using `piper-tts` with the `id_ID-news_tts-medium.onnx` model.
  - Outputs synthesized speech to `.wav` files.

## 📁 Project Structure

- `stt_live.py` - Live, real-time STT pipeline listening for a wake word.
- `stt.py` - Batch STT transcription for pre-recorded audio.
- `tts.py` - Text-to-Speech synthesis script.
- `models/` - Directory for storing local ONNX models (e.g., Piper TTS models).
- `text-from-stt/` - JSON outputs containing timestamps and transcriptions from live STT sessions.
- `tts-audio/` - Synthesized audio files generated by TTS.

## 🚀 Setup & Installation

It is recommended to run this project inside a Python virtual environment to manage dependencies securely.

1. **Create a virtual environment:**
   ```powershell
   python -m venv venv
   ```

2. **Activate the virtual environment:**
   ```powershell
   .\venv\Scripts\Activate.ps1
   ```

3. **Install dependencies:**
   ```powershell
   pip install -r requirements.txt
   ```

## 💻 Usage

Ensure your virtual environment is active before running any scripts.

### 1. Live Speech-to-Text (Wake Word Mode)
This script continuously listens to your microphone. When you say the wake word (default: "Hey Jarvis"), it starts transcribing your speech until it detects silence.

```powershell
python stt_live.py
```
*Transcriptions are saved in the `text-from-stt/` folder as JSON.*

### 2. Batch Speech-to-Text
Transcribe a specific audio file (e.g., `stt-audio/audio_protel.mp4`).

```powershell
python stt.py
```

### 3. Text-to-Speech (Indonesian)
Synthesize Indonesian text into audio. You can edit the text inside `tts.py`.

```powershell
python tts.py
```
*Outputs are saved in the `tts-audio/` folder.*
